# Flappy Bird: World Rankings

## Архитектура
- MVVM с Clean Architecture
- Room + DataStore + Fake API
- GameEngine как отдельный доменный слой

## Особенности реализации
1. Реализована стратегия "сначала кэш, потом сеть"
2. Fake API имитирует сетевые задержки и ошибки
3. Полная оффлайн-работоспособность
